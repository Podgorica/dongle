# Contributing
If you wish to contribute to this project, refer to the following standards:

## Whitespacing
> if ( x + 4 ) then

Is incorrect.

> if (x + 4) then

Is correct.

## Variable naming
> C = 3.2

Is incorrect.

> MUSIC_VOLUME = 3.2

Is correct.
