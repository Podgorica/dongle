# DONGLE 1.4
A Pong game made in Lua using the LOVE2D framework.

# Dependencies
- LOVE 2D 0.9.1
- Lua 5.1

# Comments
I first made this when I knew nothing about Lua. I still don't, really.

# Contributing
If you wish to contribute to this project, refer to the following standards:

## Whitespacing
> if ( x + 4 ) then

Is incorrect.

> if (x + 4) then

Is correct.

## Variable naming
> C = 3.2

Is incorrect.

> MUSIC_VOLUME = 3.2

Is correct.
